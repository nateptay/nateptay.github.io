What Inspired Me

To try this Library

The AOS Project on Github

When I started scrolling through Michał Sajnóg's github.io page for AOS, I recognized the idea from some high end sites. Apple uses scrolling animations for most of their site, and it has a really cool effect. Seeing the Demo page with code examples made that idea seem a whole lot more obtainable.


http://michalsnik.github.io/aos/

QuestionPro.com

This was one site I found that uses AOS well in its code. I think it had good use of white space, and the timing of the elements worked out nice so you don't have to wait too long to see information on the page. I've noticed some sites where the animated scrolling feels like it wastes the user's time, but I think this site had good pacing.


https://www.questionpro.com/

mcube.fr/projects

This site had some cool animations, particles, and layouts. I liked the project gallery animations because they keep drawing you further down the page. There's a feather running circles around the page, and animated images and text that make users want to scroll.


https://www.mcube.fr/projets/
