Kid's Animation Design

Date: 5/17/2020-5/18/2020

My Role: Entire Project

Project Type: Website Design

Languages: HTML, CSS, JavaScript

Project Demo View on GitHub

My favorite part of this site was seeing all the assets come together. I went through the design process in order this time, and loved how much easier it was to implement the design I'd envisioned by planning it out first.

This was a school project where we were given two weeks to design a website geared towards young kids.

I researched children's sites, created all the assets you see in this design, sketched out my layouts, then started coding. With midterms in other classes happening in the same time period, I did this design all in two days.
