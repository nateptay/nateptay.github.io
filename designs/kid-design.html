Kid's Animation Design

Project Title: Kid's Animation Design

Start Date: 5/17/2020

End Date: 5/18/2020

My Role: Entire Project

Program Type: Web Design

Languages Used: HTML, CSS, JavaScript

Description: This was a school project where we were given two weeks to design a website geared towards young kids. I researched children's sites, created all the assets you see in this design, sketched out my layouts, then started coding. With other classes midterms happening in the same time period, I did this design all in two days.

Highlight: My favorite part of this site was seeing all the assets come together. I went through the design process in order this time, and loved how much easier it was to implement the design I'd envisioned by planning it out first.

Purpose: The goal of this kid’s site is to engage young learners with interactive content, and spark creativity.

Target Audience: Kids, Parents of young kids, Teachers of young kids
